\doxylink{classSystem}{System} that handles sprite animation. 

Updates animation frames for entities with both \doxylink{structAnimationComponent}{Animation\+Component} and \doxylink{structSpriteComponent}{Sprite\+Component} based on elapsed time and animation parameters.
