\doxylink{classEvent}{Event} representing a mouse click action. 

Contains information about the mouse button clicked and the screen coordinates where the click occurred. Inherits from the base \doxylink{classEvent}{Event} class for use with \doxylink{classEventManager}{Event\+Manager}.
