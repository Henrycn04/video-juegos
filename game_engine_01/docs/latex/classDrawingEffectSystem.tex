\doxysubsubsection{Detailed Description}
\doxylink{classSystem}{System} that handles drawing-\/based effects on game entities. 

Processes drawing traces to apply various effects\+:
\begin{DoxyItemize}
\item Red traces damage enemies
\item Blue traces boost player speed
\item Green traces slow enemies Uses collision detection between drawing points and entities. 
\end{DoxyItemize}
