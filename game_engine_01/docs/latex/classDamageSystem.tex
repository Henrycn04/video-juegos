\doxysubsubsection{Detailed Description}
\doxylink{classSystem}{System} that handles damage application when collisions occur. 

Listens for collision events and applies damage when\+:
\begin{DoxyItemize}
\item A projectile hits an entity with health
\item The damaged entity is a player Manages player death state when health reaches zero. 
\end{DoxyItemize}
